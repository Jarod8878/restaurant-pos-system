# restaurant-pos-system
Repository for Point of Sales (POS) web application that uses React, Node.js and MySQL. Hosted on local server, with mobile phone connected to same IP address as hosting device via API_BASE.
FOH on mobile device, and BOH hosted on desktop to simulate real-world scenarios
Features included:
- Rewards and points redemption
- Inventory management with low stock alerts
- Admin and customer profile management
- Menu management (add, delete, update items)
- Order management
- Feedback management
- Sales report and inventory forecasting
